Jika Anda sudah terbiasa dengan pengembangan aplikasi mobile, Anda mungkin ingin menggunakan **React Native CLI**. Penggunaan CLI ini memerlukan Xcode (untuk iOS) atau Android Studio (untuk Android) untuk memulai.

Jika Anda sudah menginstal salah satu dari alat tersebut, Anda seharusnya bisa langsung mulai dalam beberapa menit. Namun, jika belum terinstal, siapkan waktu sekitar satu jam untuk proses instalasi dan konfigurasi.

## Daftar Isi Tutorial
Berdasarkan video di atas, berikut adalah tahapan yang akan kita lalui:

### 1. Install Node.js
React Native memerlukan Node.js. Kami menyarankan menggunakan versi LTS terbaru.
Di Ubuntu, Anda bisa menggunakan perintah berikut:

```bash
curl -sL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```

### 2. Install Java Development Kit (JDK)
Android Studio memerlukan JDK. Untuk React Native, disarankan menggunakan OpenJDK 17.

```bash
sudo apt update
sudo apt install openjdk-17-jdk
```
Cek versi untuk memastikan: `java -version`.

### 3. Download & Install Android Studio
Unduh Android Studio dari [situs resmi Google](https://developer.android.com/studio). 
Setelah diunduh, ekstrak file tersebut dan jalankan script instalasinya:

```bash
cd ~/Downloads/android-studio/bin
./studio.sh
```

### 4. Install Android SDK
Saat pertama kali menjalankan Android Studio, ikuti wizard untuk menginstal:
- Android SDK
- Android SDK Platform
- Android Virtual Device (Emulator)

Pastikan juga untuk mencentang **Android SDK Command-line Tools** di bagian SDK Manager.

### 5. Konfigurasi Environment Variable
Agar terminal mengenali alat Android, kita perlu menambahkan path ke file `.bashrc` atau `.zshrc`.

Buka file konfigurasi:
```bash
nano ~/.bashrc
```

Tambahkan baris berikut di paling bawah:
```bash
export ANDROID_HOME=$HOME/Android/Sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/platform-tools
```

Simpan (Ctrl+O, Enter) dan keluar (Ctrl+X), lalu terapkan perubahan:
```bash
source ~/.bashrc
```

### 6. Membuat Aplikasi Baru
Sekarang Anda siap membuat proyek React Native pertama Anda:

```bash
npx react-native@latest init MyFirstProject
```

Jalankan aplikasinya:
```bash
cd MyFirstProject
npm start
# (Di terminal baru)
npx react-native run-android
```
