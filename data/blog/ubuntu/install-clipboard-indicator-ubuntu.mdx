Apakah Anda sering bolak-balik menyalin teks yang berbeda? **Clipboard Indicator** adalah solusi cerdas untuk mengelola riwayat clipboard Anda. Alat ini memungkinkan Anda melihat daftar teks yang pernah Anda salin sebelumnya dan memilihnya kembali dengan satu klik.

## Langkah Instalasi

Clipboard Indicator adalah ekstensi untuk GNOME Shell (desktop default Ubuntu).

### 1. Instal Extension Manager
Agar lebih mudah mencari dan menginstal ekstensi tanpa melalui browser, kita gunakan aplikasi **Extension Manager**:

```bash
sudo apt update
sudo apt install gnome-shell-extension-manager
```

### 2. Cari & Instal Clipboard Indicator
1. Buka aplikasi **Extension Manager** yang baru saja diinstal.
2. Klik pada tab **Browse**.
3. Cari kata kunci **"Clipboard Indicator"**.
4. Klik tombol **Install** pada hasil yang muncul.

### 3. Penggunaan
Setelah terinstal, Anda akan melihat ikon kecil (berupa daftar) di pojok kanan atas taskbar Ubuntu Anda. Setiap kali Anda menekan `Ctrl + C`, teks tersebut akan tersimpan dalam riwayat yang bisa Anda akses kapan saja melalui ikon tersebut.
